Adds permalinks to headers.

== Description ==

Headerlinks adds permalinks to the headers/headings on posts and pages. When the cursor hovers over a header, a pilcrow (Â¶) symbol appears to the right of the header to indicate the permalink.

== Installation ==

1. Upload the `headerlinks/` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
